This test checks our support for the 'features' argument to window.open.

Each button below will open a new window. Check to make sure that the window has the specified attributes.

NOTE: Please verify that the behavior in this test is resolution independent.

To test a non-standard resolution:

  1. Open Quartz Debug (/Developer/Applications/Performance Tools).
  2. Select Tools -> Show User Interface Resolution.
  3. Set the resolution to 2.0.
  4. Restart Safari.

Window size (no toolbars): You should see a red 1 pixel border along every edge of this page, and no scrollbars.

Window size (all toolbars): You should see a red 1 pixel border along every edge of this page, and no scrollbars.

Window positioning: This window should be aligned exactly to the top right corner of the screen.

NOTE: This won't work with resolution 2.0 until we fix screen.width.
