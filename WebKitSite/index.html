Welcome to the web site for the WebKit Open Source Project!

WebKit is the system framework used on Mac OS X by Safari, Dashboard, Mail.app, and many other OS X applications. It is based on the KHTML engine from KDE.

Getting involved

There are many ways in which you can get involved with the project. You can:

  • check out and build the source code
  • download the latest nightly build

Once you have either of these, you can help by:

  • reporting bugs you find in the software
  • providing reductions to bugs
  • submitting patches for review

More info

More information about WebKit can be found on its wiki. You can help here too, by adding information that can help others learn about WebKit. If you have more questions, contact us.
